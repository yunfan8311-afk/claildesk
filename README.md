# ClailDesk

长风破浪会有时，直挂云帆济沧海。

## 介绍

ClailDesk 是一款开源远程桌面软件，基于 claildesk 1.2.7 开发。使用 Rust 和 Flutter 构建，开箱即用，无需配置即可使用。

## 功能特点

- ? 远程控制 - 轻松访问和控制远程计算机
- ? 文件传输 - 快速安全地传输文件
- ? 剪贴板同步 - 跨设备复制粘贴
- ? 安全可靠 - 端到端加密
- ? 自建服务器 - 支持使用自己的服务器

## 编译

### Windows

1. 安装 Rust 和 vcpkg
2. 安装 Flutter SDK
3. 克隆代码并运行：

```bash
cargo build --release
```

### 构建 Flutter 版本

```bash
cd flutter
flutter build windows
```
